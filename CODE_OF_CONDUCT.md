# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, and assume good faith. Personal attacks, harassment, or discriminatory behavior are not tolerated in issues, pull requests, or any project space.

## Reporting

If you experience or witness behavior that violates this code, report it to **hi@jitangupta.com**. Reports are handled confidentially. The maintainer will review and take whatever action is warranted — a warning, a temporary block, or a permanent ban from the project.

## Enforcement

Enforcement follows the [Contributor Covenant enforcement guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines): correction, warning, temporary ban, permanent ban — applied proportionally.
